ğŸ“‚ Types of Operating Systems

Different types of OS are used based on task handling, time sensitivity, and resource management.

  • Batch OS: Executes grouped jobs without interaction.
  • Time-Sharing OS: CPU slices are shared among users.
  • Real-Time OS: Used in robotics, medical devices.
  • Distributed OS: Controls multiple systems as one.
  • Embedded OS: Lightweight for appliances.

Shortcut Tag: Task Style + User Load + Response Time
