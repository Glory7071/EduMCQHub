⚙️ Unit 2: Process Management

🔄 Process States

  • New: Process is being created.
  • Ready: Waiting for CPU assignment.
  • Running: Executing on CPU.
  • Waiting: Awaiting an event (I/O etc).
  • Terminated: Completed execution.
“Process life cycle is like life — transitions define progress.”

🧾 Process Control Block (PCB)

The PCB contains:

  • PID, State, Program Counter
  • CPU Registers, Memory Info, I/O Info
  • Accounting info (CPU time used, etc.)

🔁 Context Switching

Switching between processes involves:

  • Saving current state to PCB
  • Loading next process from PCB
  • Overhead due to time in switching

🧠 CPU Scheduling Algorithms

  • FCFS: First-come-first-serve
  • SJF: Shortest Job First
  • Priority Scheduling
  • Round Robin: Equal time slices
  • Multilevel Queue: Different queues, different algorithms

📊 Gantt Chart Example (FCFS, SJF, RR)

FCFS: | P1 | P2 | P3 | P4 | 
       0    5    8   16   22
SJF:  | P2 | P1 | P4 | P3 | 
       0    3    8   14   22
RR:   | P1 | P2 | P3 | P4 | P1 | ...
